\pitem[Iran TST 2012 -first day- problem 1]{%
    Find all positive integers $n \geq 2$ such that for all integers $i,j$ that $ 0 \leq i,j\leq n$ , $i+j$ and $ {n\choose i}+ {n \choose j}$ have same parity.
    }{%
    Let $i = 0$, for all $0 \leqslant j = 2r \leqslant n$, ${n\choose 2r}+1$ is even, so ${n\choose 2r}$ is odd. Denote $n = {\left( {\overline {{n_t}{n_{t - 1}}...{n_1}{n_0}} } \right)_2}, 2r = {\left( {\overline {{k_t}{k_{t - 1}}...{k_1}0} } \right)_2}$, ${n_t} = 1$. By Lucas theorem $\prod\limits_{i = 0}^t {{n_i\choose k_i}} \equiv {n\choose 2r} \equiv 1(\bmod 2)$, so ${n_i\choose k_i} = 1$ for all $0 \leqslant i \leqslant t$, so ${n_0} = 0$.
    If there exit integer $1 \leqslant s \leqslant t - 1$ satisfies ${n_s} = 0$, we take $2r = {2^s}$, then ${n_s\choose k_s} = {0\choose 1} = 0$, contradiction. So $n = {\left( {\overline {11...10} } \right)_2} = {2^{t + 1}} - 2$.

    On the other hand if $n = {\left( {\overline {11...10} } \right)_2}$, because $C_1^0 = C_1^1 = C_0^0 = 1$, for any $i = {\left( {\overline {{a_t}{a_{t - 1}}...{a_1}{a_0}} } \right)_2}$, by Lucas theorem ${n\choose i} \equiv \prod\limits_{i = 0}^t {{n_i\choose a_i}} \equiv {0\choose a_i} \equiv {a_0} - 1 \equiv i-1 (\bmod 2)$, similarly we have ${n\choose j} \equiv j - 1(\bmod 2)$, so ${n\choose i} + {n\choose j} \equiv i + j(\bmod 2)$.
Above all, $n = {2^{t + 1}} - 2$, $t \in {Z^ + }$.
