\qitem{%
    Assume $a, b, c$ real distinct number such that,
    $$\left(\frac{a}{b-c}\right)^{2}+\left(\frac{b}{a-c}\right)^{2}+\left(\frac{c}{a-b}\right)^{2}=2$$and roots of $x^3-3x^2+2x+n$. Find the value of $n$.
    }{%
    Put $x=\frac{a}{b-c}$, $y=\frac{b}{c-a}$, $z=\frac{c}{a-b}$. Then the following holds. Note that $a+b+c=3$, $ab+bc+ca=2$.

    1) $xy+yz+zx=-1$

    2) $x+y+z=0$

    3) $\sum_{cyc}a(a-b)(a-c)=0$

    4) $(a+b+c)^3+9abc=4(a+b+c)(ab+bc+ca)$

    5) $abc=-\frac{1}{3}$


    Therefore, the answer is $\boxed{\frac{1}{3}}$.
