\qitem{%
    Calculate $ y $ and $ z $, so that the number $ 3y45z8 $ is divisible by $ 99 $?
    }{%
    Multiples of 11 are verified by subtracting the odd-order digits from the even-order digits:

    11: $ (8 + 5 + y) - (z + 4 + 3) \equiv 0 (11)$

    9: $ 3 + y + 5 + 4 + z + 8 \equiv 0(9)$

    Solving the system we get:
    $ (y, z) = (6,1) $

}
