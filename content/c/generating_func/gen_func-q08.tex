\qitem{%
    Let $p$ be an odd prime number. Find the number of subsets $A$ of the set $\left\{1,2,\dots ,2p\right\}$ such that
    \begin{enumerate}
        \item $A$ has exactly $p$ elements, and
        \item the sum of all the elements in $A$ is divisible by $p$.
    \end{enumerate}
