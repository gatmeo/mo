  \textbf{Double Counting}: The main idea is to count some quantity in two different ways. Often, the main difficulty of a double counting problem is figuring out the specific quantity to double count. It can be used to prove some equalities (or inequalities) or to find some bounds for the solution.
\end{mdframed}
